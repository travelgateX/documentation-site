Booking Flow

Edit page

Search, Quote and Book transactions must be executed sequentally in order to book hotel rooms in a Seller.

In the following steps you will find the three steps in order to perform the booking flow. You can also consult the Schema through our API playground besides to launch requests.

NOTE: Remember that you can configure which fields you want to receive on the response, you just have to configure them on the correspondent graph query.

Suggestions In order to help you during the booking process you can find several operations to get static data from the providers. You can find the info on our Static content page.
